Linux Basics
=============

What is Linux?
--------------
Linux is an open-source operating system. It is the underlying
operating system on which many popular operating systems like
Android, Ubuntu, Raspbian and MacOS are based on.

Getting started
---------------
Procedure
^^^^^^^^^
1. SSH into a Raspberry Pi or use a computer with Ubuntu.
2. Follow Digital Ocean's Linux crash course.
